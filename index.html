  • What is the difference between var let and const ?

    • var is a functional scope variable, can be declared without initialization.It can be updated and re-declared into the scope.
    let is a block scope variable, can be declared without initialization. It can be updated but cannot be re-declared into the scope.
    const is a block scope variable, It cannot be declared without initialization. It cannot be updated or re-declared into the scope.

  • What is the difference between arrow function and regular function?

    • Arrow functions is a new feature introduced in ES6, it is enable writing concise functions in JavaScript.
    Unlike regular functions, arrow functions do not have their own 'this'.
    Arguments objects are not available in arrow functions, but are available in regular functions.

  • What is difference between map, forEach and filter, find?

    • The map() method returns an entirely new array with transformed elements and the same amount of data.
    In the case of forEach(), even if it returns undefined, it will mutate the original array with the callback.
    Therefore, we see clearly that map() relies on immutability and forEach() is a mutator method.

    find() method return the first value which match from the collection. find() method search the array from the start if the desired value is matched then find() method return that value and terminate and rest of the array is not process.
    filter() method returns the match value from the array. it search whole array from start to end and returns all the value which is matched.
